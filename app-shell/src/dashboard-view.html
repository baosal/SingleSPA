Microfrontends in this project

This is a single-spa setup. The app-shell (root-config) owns routing and layout, and it mounts independent microfrontend applications into the page.

  • Fixed apps: always visible (e.g. sidebar/header).
  • Routed apps: mounted per route (e.g. /module-1, /module-2).
  • Each microfrontend is an independent build with its own package.json and webpack.config.js, loaded at runtime via an import map.
